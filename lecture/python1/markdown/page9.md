## 반복문 (for)

<font size="6">

- for문은 리스트와 같은 시퀀스의 아이템을 순회합니다.
``` python
>>> family = ['mother', 'father', 'gentleman']
>>> for x in family:        # family의 각 항목 x에 대하여:
...     print(x, len(x))    # x와 x의 길이를 출력하라.
...
```

- range 함수 이용
``` python
>>> for i in range(4, 8):  # [4, 5, 6, 7]
...     print(i)
...
```
</font>
