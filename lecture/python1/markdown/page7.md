## 조건문 (if-elif-else)

<font size="6">

- if: else:
``` python
>>> if a > b:                     # 만약 a가 b보다 크면
...     print('a')                # 'a'를 출력한다.
... else:                         # 그렇지 않으면
...     print('b')                # 'b'를 출력한다.
... 
```

- elif
``` python
>>> c = 15 * 5
>>> d = 15 + 15 + 15 + 15 + 15
>>> if c > d:
...     print('c is greater than d')
... elif c == d:
...     print('c is equal to d')
... elif c < d:
...     print('c is less than d')
... else:
...     print('I don\'t know')
... 
c is equal to d
```
